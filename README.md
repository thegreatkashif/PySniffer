# PySniffer

PySniffer is a Python-based network packet analyzer built for learning packet capture, protocol analysis, and defensive network monitoring.

## Features

- Live packet capture
- Protocol detection
- Packet filtering
- Statistics
- JSON export
- CSV export
- Colored terminal output
- Unit tests
- GitHub Actions CI

## Technologies

- Python
- Scapy
- Colorama
- Pytest
- GitHub Actions
